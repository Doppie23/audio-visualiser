# Audio Visualiser

A desktop audio visualiser. Currently Windows only.

![demo](/img/gif.gif)

## Dependencies

- Zig `0.14.1`
- Raylib

## Features

- Spectrum Analyser
- Stereometer
  - Like the one that can be found on the [MSD200](https://www.hansens.dk/pages/documentation/msd/MSD200_Manual.pdf) (see page 15).
- Waveform

## Getting Started

```system
$ zig build run
```
